# ⚡ Smart Energy Calc - Consumo Inteligente

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
![GitHub](https://img.shields.io/badge/github-%23121011.svg?style=for-the-badge&logo=github&logoColor=white)

Este projeto é uma ferramenta simples e eficiente desenvolvida em **Python** para estimar o consumo de energia elétrica mensal de aparelhos domésticos.

## 🚀 Objetivo
Ajudar o usuário a ter controle sobre os gastos de energia, calculando o consumo em kWh e o custo estimado em Reais.

## 🧮 Fórmula Utilizada
Para chegar ao resultado, o sistema utiliza a seguinte lógica:

$$Consumo (kWh) = \frac{Potência (W) \times Horas \times 30}{1000}$$

## 📈 Exemplo de Teste (Chuveiro Elétrico)
Para validar o sistema, utilizamos os seguintes dados de um chuveiro comum:
- **Potência:** 5500W
- **Uso diário:** 0.5h (30 minutos)
- **Resultado:** 82.50 kWh/mês
- **Custo Estimado:** R$ 61,88 (Tarifa base: R$ 0,75/kWh)
